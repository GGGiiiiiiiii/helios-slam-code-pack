## 已具备（无需再提供，供对照）

| 项目 | 话题 / 接口 | 状态 |
|------|-------------|------|
| 前雷达 | `/sr_amr_control/front/scan`（LaserScan，约 5Hz，覆盖 ≥333°） | 真机已验证 |
| 后雷达 | `/sr_amr_control/rear/scan` | 真机已验证 |
| **原生里程计** | `/sr_amr_control/odom`（`nav_msgs/Odometry`，约 3.33Hz，v1.3.0） | 真机可用（外参无关，直接融合） |
| **IMU** | `/camera/gyro_accel/sample`（**头部** Orbbec Gemini 330，约 200Hz） | 数据可用；外参在 head/相机链，非底盘 TOF |
| 速度控制 | `/sr_amr_control/remote_control_cmd_vel` + 远程使能服务 | 可用 |
| 系统状态 | `/sr_amr_control/system_state`（含速度、急停等） | 可用 |
| 雷达开关 | `/sr_amr_control/lidar_enabled`（SetBool） | 可用 |

> 雷达无效距离用 `101.0` 表示无回波，导航侧会过滤，无需改 SDK。
> IMU 需单独启动 Orbbec 相机：`ros2 launch orbbec_camera gemini_330_series.launch.py enable_accel:=true enable_gyro:=true enable_sync_output_accel_gyro:=true`（导航 launch 需一并拉起或保持相机常开）。
