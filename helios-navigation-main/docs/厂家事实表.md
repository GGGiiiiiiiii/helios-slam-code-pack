## 6. 整机 URDF 关节链（厂家提供，用于接头部 IMU）

厂家整机 URDF：`/home/admin/cyh/wheeled_robot_pkg/install/share/robot_description/urdf/wheel_robot.urdf`

```
world └ chassis_base_link
        ├ chassis_left/right_Link（驱动轮）
        └ vtrunk_base_link └ trunk_base_link
              ├ trunk1..4_link（躯干）
              └ head1/head2_link（头部，Orbbec/IMU 挂此链）
        chassis_base_link
              ├ vleft_arm_link  → ... → Lhand_base_link
              └ vright_arm_link → ...
```

> 本导航工程只用底盘，简化 URDF 未含 trunk/head/臂链。真机开 EKF 用头部 IMU 时，需把 `base_footprint → chassis → trunk → head → camera/imu` 这条 TF 补进来（或另发静态外参）。
